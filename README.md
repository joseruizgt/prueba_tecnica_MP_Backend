## Run Locally

Clone the project

```bash
  git clone https://github.com/joseruizgt/prueba_tecnica_MP_Backend
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
